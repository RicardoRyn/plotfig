    快速出图¶

    脑区图使用了surfplot的API，结合图集文件，实现对某块脑区值的绘制。

    目前支持图集包括：

    1. 人 Glasser图集1
    2. 人 BNA图集2
    3. 猕猴 CHARM 5-level图集3
    4. 猕猴 CHARM 6-level图集3
    5. 猕猴 BNA图集4
    6. 猕猴 D99图集5
    7. 黑猩猩 BNA图集6

    例如绘制人类Glasser图集中左脑V1区域值为1，右脑MT区域值为1.5。

    参数设置¶

    全部参数见plotfig.brain_surface的API 文档。

    Note

    人/黑猩猩/猕猴脑区图绘制参数相同，只是函数名称以及图集名称不同。

    我们可以选择真实沟回surface作为underlay图。 同时选择surfplot api自带的colorbar (功能相对较少)。

    from plotfig import *

data = {"lh_V1": 1, "rh_MT": 1.5, "rh_V1": -1}

fig = plot_human_brain_figure(
    data,
    surf="midthickness",
    atlas="glasser",
    cmap="bwr",
    vmin=-1,
    vmax=1,
    colorbar=True,
    colorbar_label_name="A",
    colorbar_fontsize=10,
    colorbar_nticks=4,
    colorbar_shrink=0.2,
    colorbar_location="right",
    colorbar_draw_border=True,
)

    也可以使用自定义功能更多的rjx_colorbar。

    Warning

    使用rjx_colorbar时请确保colorbar为False。

    from plotfig import *

data = {"lh_V1": 1, "rh_MT": 1.5, "rh_V1": -1}

fig = plot_human_brain_figure(
    data,
    surf="inflated",
    atlas="glasser",
    cmap="viridis",
    vmin=-1,
    vmax=1,
    colorbar=False,
    rjx_colorbar=True,
    rjx_colorbar_direction="vertical",
    rjx_colorbar_label_name="A",
    rjx_colorbar_tick_length=3,
    rjx_colorbar_label_fontsize=15,
    rjx_colorbar_tick_fontsize=10,
    rjx_colorbar_outline=True,
    rjx_colorbar_nticks=2
)

    rjx_colorbar还允许水平方向的colorbar，更加节省图片空间。

    from plotfig import *

data = {"lh_V1": 1, "rh_MT": 1.5, "rh_V1": -1}

fig = plot_macaque_brain_figure(
    data,
    surf="veryinflated",
    atlas="charm5",
    cmap="viridis",
    vmin=-1,
    vmax=1,
    colorbar=False,
    rjx_colorbar=True,
    rjx_colorbar_direction="horizontal",
    rjx_colorbar_label_name="A",
    rjx_colorbar_tick_length=3,
    rjx_colorbar_label_fontsize=15,
    rjx_colorbar_tick_fontsize=10,
    rjx_colorbar_outline=True,
    rjx_colorbar_nticks=3
)

    半脑¶

    有时我们只希望展示半边大脑。

    Note

    绘制半脑的函数参数与全脑基本相同。但是多了一个hemi来指定绘制的半边，且仅有普通的colorbar，而没有rjx_colorbar。 需要强调的是，绘制左脑，请保证脑区名字为“lh_”开头，同理，绘制右脑，请保证脑区名字为“rh_”开头。

    from plotfig import *

data = {"lh_V1": 1}

fig = plot_human_hemi_brain_figure(
    data,
    hemi="lh",
    title_name="Left brain V1"
)


    from plotfig import *

data = {"rh_V1": 1}

fig = plot_human_hemi_brain_figure(
